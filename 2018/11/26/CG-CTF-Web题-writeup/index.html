$md51 = md5('QNKCDZO');
$a = @$_GET['a'];
$md52 = @md5($a);
if(isset($a)){
if ($a != 'QNKCDZO' && $md51 == $md52) {
    echo "nctf{*****************}";
} else {
    echo "false!!!";
}}
else{echo "please input a";}

利用PHP弱类型，MD5加密QNKCDZ0发现是0Exxxxxxxxx之类的字符，弱类型比较表示0
只要找到md5加密后字符串以0e开头即可
